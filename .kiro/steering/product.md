# Product Overview

QwenCoder Mac Studio is a native macOS (Apple Silicon) desktop application for local AI-assisted coding. It bundles a local MLX vision server with an Electron-based IDE that supports:

- Local LLM inference via MLX on Apple Silicon (no cloud dependency)
- Vision model support for image analysis
- Agentic coding with tool use (file read/write, bash, search, browser automation via Playwright)
- Spec-driven development workflow (requirements → design → tasks → implementation)
- Task graph orchestration with DAG-based execution, branching, and looping
- Multi-agent architecture with specialized subagent types (explore, context-gather, code-search, implementation, etc.)
- Project/session management with conversation history persistence
- Git integration, file watching, and live preview
- Conversation compaction via claw-compactor
